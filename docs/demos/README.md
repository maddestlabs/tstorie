## Usage Examples

**Single demo:**
```
https://maddestlabs.github.io/tstorie/?demo=clock
```

**With custom font:**
```
https://maddestlabs.github.io/tstorie/?demo=rainclock&font=Roboto+Mono
```

**With shader effect:**
```
https://maddestlabs.github.io/tstorie/?demo=canvas_demo&shader=SHADER_GIST_ID
```

> **Note:** The `.md` extension is optional - it will be added automatically if not present.

## Loading from GitHub Gist

You can also load custom content from GitHub Gist:
```
https://maddestlabs.github.io/tstorie/?gist=YOUR_GIST_ID
```

This is useful for sharing custom examples or external content without adding them to the repository.
