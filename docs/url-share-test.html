3. How to Use

Manual Usage:

You can also use these functions in the browser console on the main t|Storie page:

  • generateShareableUrl(content) - Generate a shareable URL
  • copyShareableUrl(content) - Generate and copy to clipboard

URL Format:

https://your-tstorie-url/?content=decode:COMPRESSED_DATA

Browser Support:

Requires modern browsers with CompressionStream API support (Chrome 80+, Edge 80+, Safari 16.4+)
